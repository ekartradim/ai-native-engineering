How to Create a Skill
Teach Claude once. It replicates forever.
1
Define intent
What should this skill enable Claude to do? When should it trigger? Clear trigger description prevents undertriggering.
2
Write SKILL.md
Instructions the agent reads on trigger. Under 500 lines. Add scripts/ for reusable code, assets/ for templates.
3
Write test cases
2 to 3 realistic, messy prompts. Write how a real user would ask, not a sanitised ideal.
4
Run evals in parallel
One agent runs with the skill, one runs without. Grade the diff. Repeat per test case simultaneously.
5
Iterate until satisfied
Read feedback, improve SKILL.md, rerun. Generalise fixes across all prompts, not just the failing one.
↻ loops back to step 4
6
Package and ship
Run package_skill.py to validate and zip. Produces a .skill file ready to install.
A skill is a replicator block. Teach it once, it assembles itself into every future task.
