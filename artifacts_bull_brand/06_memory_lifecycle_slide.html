/ce-compound-refresh Maintain

Docs rot. Stale learnings mislead — worse than no learnings. 5-phase maintenance cycle:

1
Investigate each learning against current codebase
2
Check pattern docs for overlap & contradictions
3
Classify — Keep / Update / Consolidate / Replace / Delete
4
Execute — actually edits, merges, and deletes (doesn't just flag)
5
Commit changes to git

Update

Fixes paths, class names, snippets in-place

Consolidate

Merges overlapping docs into one canonical version

Replace

Spawns subagent to rewrite, deletes old

Delete

Removes file — git history is the archive

Smart triggering
New fix contradicts existing learning New fix supersedes older solution Refactor/rename likely invalidated older docs Related Docs Finder surfaced stale candidates
